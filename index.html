Experiencia

Página Web de Lealtad – Frontend
2025

La Página Web de Lealtad es un proyecto en curso orientado al desarrollo de una plataforma digital de fidelización de usuarios. En este trabajo estoy a cargo del frontend, utilizando Angular, TypeScript, SCSS y Bootstrap para construir una interfaz moderna, responsiva y alineada con las necesidades del proyecto.

La aplicación inicia con un splash screen que muestra el logotipo de la plataforma, redirigiendo posteriormente a los módulos de inicio de sesión y registro. Dentro de la solución se han integrado diversos servicios clave como recuperación de contraseña, mensajería a soporte, así como la consulta de términos y condiciones, todos conectados a través de endpoints proporcionados por el equipo de backend.

Una vez autenticado, el usuario puede acceder a beneficios personalizados, información de perfil y recursos exclusivos, reforzando la propuesta de valor de la plataforma. Este proyecto me ha permitido profundizar en el consumo de servicios REST, la organización de componentes en Angular y la implementación de prácticas de diseño escalables en aplicaciones web profesionales.

App de Gifs
2025

La App de Gifs fue un proyecto personal desarrollado con el objetivo de fortalecer mis conocimientos en el framework Angular y el lenguaje TypeScript. La aplicación consistía en un buscador de gifs que permitía al usuario consultar contenido en tiempo real mediante la integración de una API de gifs. Además, incluía un panel lateral que mostraba tanto el usuario como el historial de búsquedas, facilitando la navegación y el reuso de consultas previas.

Para la construcción de la interfaz se emplearon HTML, TypeScript y la librería de estilos Tailwind CSS, lo que permitió diseñar una UI moderna, responsiva y ligera. Este proyecto me brindó la oportunidad de practicar la arquitectura de componentes en Angular, el consumo de APIs externas y el manejo del estado de la aplicación, reforzando mis habilidades en el desarrollo frontend con tecnologías modernas.

App Entre Páginas – Frontend
2025

La App Entre Páginas fue un proyecto académico orientado al desarrollo de un e-commerce especializado en la venta y consulta de libros. El trabajo se organizó bajo la metodología ágil Scrum, con el apoyo de Jira para la gestión y seguimiento de tareas, y Figma para la creación de los mockups iniciales que definieron la estructura y la experiencia de usuario.

El equipo se dividió en dos áreas: frontend y backend. En mi rol dentro del frontend, participé en la construcción de la interfaz de usuario empleando tecnologías como React.js, Bootstrap, JavaScript, HTML y CSS, asegurando un diseño responsivo y una experiencia de navegación clara e intuitiva. El backend fue desarrollado en Python, encargado de la creación de endpoints y la integración con múltiples APIs de libros, mientras que en la gestión de datos y autenticación se utilizó Firebase.

Este proyecto me permitió fortalecer mis habilidades en el consumo de APIs, el trabajo colaborativo entre equipos con responsabilidades diferenciadas, y la implementación de un flujo completo de un e-commerce, desde la definición visual mediante mockups hasta la comunicación con los servicios de backend.

Sitio informativo Colegio Elie Vinet
2024

El sitio informativo del Colegio Elie Vinet fue un proyecto académico desarrollado en equipo de dos personas bajo la metodología ágil Scrum, apoyándonos en herramientas como Jira y Microsoft Teams para la gestión y comunicación. El objetivo del sitio fue brindar a la comunidad escolar un espacio digital donde se pudiera consultar información relevante sobre la institución, incluyendo misión, visión y valores, detalles sobre colegiaturas, calendario de asambleas y un apartado tipo blog para anuncios y noticias. Se implementó además un módulo de registro e inicio de sesión, que permitía a los usuarios acceder a contenido exclusivo una vez autenticados.

Para el diseño previo, elaboramos los mockups en Wix, lo que facilitó la validación temprana de la estructura visual y la experiencia de usuario. En el desarrollo final empleamos tecnologías como React.js, JavaScript, HTML, CSS y Bootstrap en el frontend, junto con Firebase para la gestión de base de datos y autenticación. Este proyecto me permitió reforzar competencias en el trabajo colaborativo bajo un marco ágil, el desarrollo de interfaces informativas y seguras, y la integración de sistemas de autenticación para mejorar la experiencia del usuario.

Sitio informativo Eccoinnova
2024

El sitio informativo Eccoinnova fue un proyecto académico de la materia de Integradora en el que desarrollamos una página web para presentar el producto Tinaco LIAJJ, un sistema innovador de recolección de agua pluvial mediante compuerta y filtración. El sitio tenía como objetivo informar sobre las características del producto y del equipo de trabajo, e incluía secciones como Inicio, Nosotros, Producto y un módulo de Iniciar Sesión/Registro. Para la parte de diseño previo, elaboramos los mockups en Wix, lo que nos permitió definir la estructura y la experiencia de usuario antes de implementar el desarrollo. En la construcción final se emplearon tecnologías como React.js, JavaScript y Bootstrap en el frontend, mientras que en el backend se utilizó Firebase como servicio de base de datos y autenticación. Este proyecto me permitió fortalecer habilidades en el diseño de interfaces responsivas, la integración de servicios externos y el trabajo en equipo, además de concientizarme sobre la importancia de proponer soluciones tecnológicas enfocadas en el aprovechamiento de recursos naturales.
